ABOUT / RUNTIME ARCHITECTURE

One face.
One deformation surface.

This app bakes Google GNM Head v3 offline with the population-mean identity and 51 named facial targets. Browser playback maps waveform-refined speech gestures directly into that expanded native basis every frame—without a learned animation model.
